# VS4Mac App Center

**App Center** is a Visual Studio for macOS addin to manage Apps in Visual Studio App Center:
- Manage Apps (list, create, update or delete).
- List prganizations.
- Access to app analytics (audience, sessions, etc).
- Manage tests.
